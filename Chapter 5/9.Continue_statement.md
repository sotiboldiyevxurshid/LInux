# Continue bayonot

#### Continue iborasi FOR, WHILE yoki UNTIL siklining keyingi iteratsiyasini davom ettirish uchun ishlatiladi.


# Sintaksis

```
continue
```

### yoki

```
continue n
```

### yoki

```
...
..
for i in something
do
	[ condition ] && continue
	cmd1
	cmd2
	
done
..
...
```

#### yoki


```
...
..
while true
do
	[ condition1 ] && continue
	cmd1
	cmd2
	[ condition2 ] && break
done
..
...
```
